# リリースノート v0.7.0

リリース日: 2025年5月17日

## 🚀 主な機能追加・改善

### マークダウンエディタの進化
- **数式表示機能 (ec99381)**
    - `remark-math`と`rehype-katex`を導入し、LaTeX形式の数式を美しくレンダリング。
    - 関連するCSSファイルをインポートし、数式表示のスタイルを最適化。
- **マインドマップ機能 (4ea9ad0, 4bb572f, 0d52940)**
    - `markmap-lib`と`markmap-view`を導入し、Markdownからインタラクティブなマインドマップを生成。
    - ``` ```markmap ``` 記法でマークダウンの見出し構造を視覚的に表現。
    - ツールバーにマインドマップボタンを追加し、簡単に挿入可能に。
    - ダークモード対応を強化し、テーマに合わせた表示を実現 (4de3f3d)。
- **Marpプレゼンテーション機能強化 (7951569)**
    - スライドのスクロール位置を保存・復元する機能を追加。
    - スライドに一意のIDを付与し、ナビゲーションを改善。

### AIチャット機能拡張
- **ファイル添付機能 (27b9f44, f75685f, 8063a1d)**
    - PDF、画像、オフィスファイルなどを添付してAIに解析させる機能を追加。
    - Markitdownツールを統合し、多様なファイル形式をMarkdownに変換。
    - ファイルデータを適切な形式（data URL）に変換する処理を追加。
- **UIとユーザー体験の改善**
    - アコーディオン機能を追加し、長いメッセージを折りたたんで表示 (bca1a64, 2b04ca9)。
    - メッセージの最初の数行をプレビュー表示する機能を実装。
    - プレースホルダーを更新し、エディタ内容参照機能を明示 (0c9d8ce)。
- **テキスト選択と操作機能 (978584b)**
    - 選択したエディタのテキストを取得・置換する機能を追加。
    - エディタとチャットの連携を強化し、より直感的な操作を実現。
- **Ollamaプロバイダー改善 (116fad1)**
    - ストリーミング設定を修正し、ファイル添付時の処理を最適化。

## 🔧 依存関係の更新・整理
- `remark-math`, `rehype-katex`, `katex` を追加 (ec99381)
- `markmap-lib`, `markmap-view` を追加 (4ea9ad0)
